## ✅ 9-MINUTE SETUP

### Step 1: Sign Up (2 min)
```
https://www.brevo.com/free-email
→ Click "Sign up for free"
→ Verify email
→ Done! Account created
```

### Step 2: Get API Key (2 min)
```
https://app.brevo.com
→ Settings → SMTP & API → API Tokens
→ Create new token
→ Copy token (starts with xkeysib-)
```

### Step 3: Configure Render (2 min)
```
https://render.com/dashboard
→ TredingSystem service → Environment
→ Add: EmailProvider__Type = Brevo
→ Add: EmailProviders__Brevo__ApiKey = xkeysib-...
→ Save
```

### Step 4: Wait (3 min)
```
→ Render deploys with new settings
→ Watch for: "Deploy Succeeded" ✅
```

### Step 5: Test (1 min)
```
→ Sign up at: https://tredingsystem.onrender.com
→ Check email inbox
→ Verify OTP works
→ Done! ✅
```

**Total: 9 minutes from nothing to working email! 🎉**
